# 3D Sci-Fi Game DemoThis 3D Sci-Fi game demo was developed with Unity and C\#, by following the [tutorial](https://www.udemy.com/the-ultimate-guide-to-game-development-with-unity/) on Udemy.Download the demo video [here](https://github.com/Jw-Jn/Sci-Fi-Game-Demo/blob/master/demo.mp4?raw=true).## How to play the game* Double click "release/Sci-Fi Demo.exe" to start the game.* Press w/s/a/d to control the character to move in the scene.* Press "e" to pick up the coin or use the coin to buy weapon.* Click left mouse button to shoot once you have a weapon.* Press "r" to reload bullets.* Press "Win"+"d" to exit the full screen.![image](demo.gif)
